## Request/Response Flow

### Game Image Selection

```yaml
1. User clicks image card
в””в”Җв–ә ImageSelector.handleImageSelect()
в””в”Җв–ә useGameDataStore.selectImage(imageId)
в””в”Җв–ә navigate("/")

2. Index page detects selectedImageId
в””в”Җв–ә Renders GameImage component
в””в”Җв–ә useSingleImageQuery(imageId) [React Query]
в””в”Җв–ә getImageById(imageId) [Axios]
в””в”Җв–ә GET /api/image/:id

3. Response validation
в””в”Җв–ә ImageDataSchema.parse(response.data)
в””в”Җв–ә useGameDataStore.setSelectedImageData(data)
```

### Character Finding

```yaml
1. User clicks on game image
в””в”Җв–ә GameImage onClick handler
в””в”Җв–ә useTargetBox.handleImageClick(x, y)
в””в”Җв–ә useCharacterPositions recalculates tolerance

2. Target box shows with character list
в””в”Җв–ә CharacterList renders notFoundCharacters
в””в”Җв–ә useGameProgressStore

3. User selects character from dropdown
в””в”Җв–ә TargetBox.onCharacterClick(characterName)
в””в”Җв–ә useGameProgressStore.markCharacterAsFound(name)
в””в”Җв–ә If all found вҶ’ areAllCharactersFound() = true
в””в”Җв–ә GameActions.endGame()
в””в”Җв–ә useTimerStore.stop()
в””в”Җв–ә useGameStatusStore.setStatus("ENDED")
в””в”Җв–ә WinnerForm displayed
```

### Score Submission

```yaml
1. User completes form
в””в”Җв–ә WinnerForm.handleSubmit()
в””в”Җв–ә useCompletionMutation()
в””в”Җв–ә createGameCompletion(data, imageId) [Axios]
в””в”Җв–ә POST /api/image/:id/game-completion

2. Backend validates
в””в”Җв–ә GameCompletionSchema.parse(body)
в””в”Җв–ә Verify image exists
в””в”Җв–ә prisma.gameCompletion.create()

3. Success response
в””в”Җв–ә React Query invalidates leaderboard cache
в””в”Җв–ә navigate("/leaderboard/:id")
в””в”Җв–ә useLeaderboardQuery(id)
в””в”Җв–ә GET /api/image/:id/leaderboard
```

## Data Architecture

### Client-Side State (Zustand)

| Store          | Purpose             | Key State                               |
| -------------- | ------------------- | --------------------------------------- |
| `gameData`     | Selected image info | `selectedImageData`, `aspectRatio`      |
| `gameStatus`   | Game lifecycle      | `status: IDLE\|RUNNING\|PAUSED\|ENDED`  |
| `gameTimer`    | Elapsed time        | `seconds`, `isRunning`, `isPaused`      |
| `gameProgress` | Found characters    | `foundCharacters`, `notFoundCharacters` |
| `gameUI`       | Overlay state       | `boxPosition`, `showInfoModal`          |
| `gameMetrics`  | Final metrics       | `secondsTaken`, `startTime`             |

### Server-Side State (React Query)

| Query                 | Key                        | Stale Time |
| --------------------- | -------------------------- | ---------- |
| `useImagesQuery`      | `["images"]`               | 24 hours   |
| `useSingleImageQuery` | `["image", id]`            | 24 hours   |
| `useLeaderboardQuery` | `["leaderboard", imageId]` | 5 minutes  |
